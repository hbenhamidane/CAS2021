analysis module

Created on 07 Sep 2021

@author = “Ludovic Le Reste” @credits = [“Hisham Ben Hamidane”, “Ludovic Le Reste”] @status = “Prototype”

This scripts loads and analyses the Waterbase WISE 6 database from the European Environment Agency. Analysis was limited to data from Switzerland Data sources Measurements, disaggregated data:

https://discomap.eea.europa.eu/App/DiscodataViewer/?fqn=[WISE_SOE].[v1r1].[Waterbase_T_WISE6_DisaggregatedData]

Spatial Metadata:

https://discomap.eea.europa.eu/App/DiscodataViewer/?fqn=[WISE_SOE].[v1r1].[Waterbase_S_WISE_SpatialObject_DerivedData]
