analysis_EU¶

Created on 28 June 2022

@author = “Ludovic Le Reste” @credits = [“Hisham Ben Hamidane”, “Ludovic Le Reste”] @status = “Prototype”

This scripts loads and analyses the Waterbase WISE 6 database from the European Environment Agency. Analysis is for the whole dataset (not limited to Switzerland as previously) Data sources: 2021 WISE dataset (disaggregated data, aggregated data, spatial data):

https://www.eea.europa.eu/data-and-maps/data/waterbase-water-quality-icm-2
